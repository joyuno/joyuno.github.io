오늘은 **면접 준비 자료 정렬**부터 **트레이딩 봇 백테스트**와 **블로그 SEO 설정**까지, AI에게 **구조화된 출력 형식**을 미리 지정하는 패턴으로 하루를 보냈다. 단순 질문을 던지는 대신 여러 MD 파일로 나누어 달라고 요청하니 출력이 체계적이었고, bit-scalp 프로젝트에서 특히 **지표 상관관계 분석** 프롬프트가 빛을 발했다.

## 면접 자료 MD 파일로 구조화
kaya.ai 건설 PM 자동화 면접을 위해 회사·직무 연구와 영어 스크립트를 요청했다. 이전처럼 "회사 알려줘" 식으로 하면 산만해지던 걸, **여러 MD 파일로 나눠달라**고 명시하니 깔끔하게 정리됐다.

> "1번부터 작성해줘 그리고 정확히 내가 뭐하는 직무인지 부터 물어보고 시작해야해서 [...] 여러 md 파일로 정리해줘"

**잘 통한 점**: 파일명 지정(`00_overview.md`, `01_company.md` 등)과 **순차 번호 매김**으로 AI가 로드맵을 따라 응답. Suffolk Construction 한 문장 요약부터 포트폴리오 프로젝트 영어 1분 스크립트까지 완성도 높음.

**안 통한 점**: 내 추정 직무(서울 재택+부산 출장)를 그대로 반영해 **가정 과다** 경고가 나옴. 실제 공고 링크를 안 줘서 일반론으로 흘렀다.

노하우:
1. 프롬프트 끝에 `mkdir`으로 폴더 생성 제안 유도 — 실제로 써먹음.
2. 출력 후 **인터뷰어 LinkedIn** 언급처럼 후속 연구 아이디어 자동 제안.
3. 검증: `02_opening.md` 스크립트 직접 읽어보니 자연스러움 90%.

AI 한계: Amber 데모 동영상 시연은 텍스트로만 설명 — 직접 봐야 한 부분.

## 블로그 SEO 등록 가이드
joyuno.github.io를 Google Search Console과 Analytics에 등록하려 했는데, AI가 **단계별 curl 명령**까지 제안해줬다.

> "현재 Google Search Console 등록해서 joyuno.github.io 를 검색 엔진에 등록하고 가능하면 애널리틱스까지 연동해서 [...] 어떻게하면될까"

**잘 통한 점**: GitHub Pages 특화 컨텍스트(joyuno 깃헙 주소 언급) 주니 `echo`와 `curl` 카테고리 명령으로 즉시 실행 가능 가이드.

| 옵션 | 장점 | 단점 |
|------|------|------|
| Search Console | 검색 노출 즉시 | 검증 파일 업로드 필요 |
| Analytics | 트래픽 추적 | GA4 설정 복잡 |

**안 통한 점**: "joyuno 검색 안 나옴" 증상을 재현 테스트 제안 없음 — 직접 `git log`로 커밋 확인해야.

**배운 점**: GitHub 도메인 입력 시 **커스텀 도메인 vs Pages 차이** 자동 설명. 다음엔 스크린샷 컨텍스트 추가.

## bit-scalp 백테스트 지표 조합
RSI/StochRSI 후행성 문제로 손실 나는 봇 전략을 업그레이드. **상관관계 낮은 지표 조합 생성+백테스트** 요청이 핵심이었고, 3일 데이터(2026-05-10~12)로 돌렸다.

> "Binance USDT-M Futures에서 가용한 즉각 반영 지표/데이터 전부 나열 [..] 각 조합을 다양한 시장 상황 시기로 백테스트"

**잘 통한 점**: 시장 상황(횡보/조정/상승)별 분류 프롬프트로 `backtest_3day.py` 생성. 초기 잔고 $290, 10x 레버리지 등 **현실 파라미터** 반영해 신뢰성 ↑.

**안 통한 점**: Node/Python 믹스 환경(`node`, `python3`, `pip`)에서 `uv`나 `source` 충돌 — AI가 env 세팅만 제안, 직접 `ls`/`find`로 디버깅.

테스트 결과 요약:
- StochRSI 단독: 횡보장 -15% 손실
- 상관 낮은 조합(RSI+Volume): +8% 수익 (3일 상위 200종)

노하우: 프롬프트에 **백테스트 환경 상세(슬롯 분할, 수수료)** 넣으니 코드가 즉전. Cloudflare CLI 피드백처럼 "적용 안 돼" 불만 시 **대안 스크립트(`deploy-store.sh`)** 즉시 생성.

AI 한계: 실제 데이터 다운로드(`pip` 패키지)는 안 해줌 — 분업: AI 설계, 나 실행.
