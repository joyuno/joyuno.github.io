1. NessusлһҖ?

NessusлҠ” Tenableмқҙ к°ңл°ңн•ң м—…кі„ мөңкі мқҳ м·Ём•Ҫм җ мҠӨмәҗл„ҲлЎң, м „ м„ёкі„ мЎ°м§Ғм—җм„ң л„ӨнҠёмӣҢнҒ¬, мӢңмҠӨн…ң, м• н”ҢлҰ¬мјҖмқҙм…ҳмқҳ ліҙм•Ҳ м·Ём•Ҫм җмқ„ мӢқлі„н•ҳкі  нҸүк°Җн•ҳлҠ” лҚ° мӮ¬мҡ©лҗ©лӢҲлӢӨ. AI л°Ҹ лҚ°мқҙн„° мқён”„лқј нҷҳкІҪм—җм„ңлҸ„ н•„мҲҳм Ғмқё лҸ„кө¬мһ…лӢҲлӢӨ.

3лҢҖ н•өмӢ¬ лҠҘл Ҙ

рҹ”Һ м·Ём•Ҫм җ нғҗм§Җ

150,000к°ң мқҙмғҒмқҳ н”Ңлҹ¬к·ёмқёмқ„ нҶөн•ҙ OS, лҜёл“ӨмӣЁм–ҙ, лҚ°мқҙн„°лІ мқҙмҠӨ, м»Ён…Ңмқҙл„Ҳ нҷҳкІҪмқҳ м·Ём•Ҫм җмқ„ м •л°Җ м§„лӢЁн•©лӢҲлӢӨ.

рҹ“Ӣ к·ңм • мӨҖмҲҳ к°җмӮ¬

CIS, DISA STIG, PCI DSS, GDPR л“ұ мӮ°м—… л°Ҹ к·ңм ң н‘ңмӨҖм—җ лҢҖн•ң мһҗлҸҷнҷ”лҗң м»ҙн”Ңлқјмқҙм–ёмҠӨ м җкІҖмқ„ м ңкіөн•©лӢҲлӢӨ.

REST API, CLI, Python лқјмқҙлёҢлҹ¬лҰ¬(tenable)лҘј нҶөн•ҙ CI/CD нҢҢмқҙн”„лқјмқё л°Ҹ AI мӣҢнҒ¬лЎңл“ңм—җ м·Ём•Ҫм җ мҠӨмә”мқ„ нҶөн•©н•©лӢҲлӢӨ.

2. мҷң мҚЁм•ј н•ҳлҠ”к°Җ?

NessusлҠ” лӢЁмҲңн•ң мҳӨн”ҲмҶҢмҠӨ мҠӨмәҗл„ҲлҘј л„ҳм–ҙ, м—”н„°н”„лқјмқҙмҰҲ к·ңлӘЁмқҳ м •нҷ•м„ұкіј нҷ•мһҘм„ұмқ„ м ңкіөн•©лӢҲлӢӨ. м•„лһҳ л№„көҗн‘ңлҘј нҷ•мқён•ҳм„ёмҡ”.

4.3 Python SDKлҘј мӮ¬мҡ©н•ң мһҗлҸҷнҷ” мҠӨмә”

tenable лқјмқҙлёҢлҹ¬лҰ¬лҘј нҷңмҡ©н•ң мҳҲм ң:

import os
from tenable.io import TenableIO

# нҷҳкІҪ ліҖмҲҳм—җм„ң API нӮӨ лЎңл“ң (к¶ҢмһҘ)
access_key = os.environ.get('TIO_ACCESS_KEY')
secret_key = os.environ.get('TIO_SECRET_KEY')

tio = TenableIO(access_key, secret_key)

# мғҲ мҠӨмә” мғқм„ұ
scan = tio.scans.create(
    name='AI мқён”„лқј м •кё° мҠӨмә”',
    description='GPU л…ёл“ң л°Ҹ MLflow м„ңлІ„ м·Ём•Ҫм җ м җкІҖ',
    targets=['10.0.1.50', '10.0.1.60'],
    template='advanced'
)

# мҠӨмә” мӢӨн–ү & лҢҖкё°
scan.launch()
scan.wait()  # мҷ„лЈҢк№Ңм§Җ лё”лЎңнӮ№

# кІ°кіј мҡ”м•Ҫ м¶ңл Ҙ
for finding in scan.results():
    print(f"{finding['plugin_id']}: {finding['severity']} - {finding['plugin_name']}")

вҡ пёҸ мЈјмқҳ: API нӮӨлҠ” м ҲлҢҖ мҪ”л“ңм—җ н•ҳл“ңмҪ”л”©н•ҳм§Җ л§Ҳм„ёмҡ”. нҷҳкІҪ ліҖмҲҳ лҳҗлҠ” VaultлҘј мӮ¬мҡ©н•ҳм„ёмҡ”.
